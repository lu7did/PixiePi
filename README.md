PixiePi
=======

Ham radio Pixie Transceiver controlled by a Raspberry Pi Zero W board


Requirements
sudo apt-get install i2c-tools libi2c-dev
git clone git://git.drogon.net/wiringPi

Build

cd /home/pi/PixiePi/src
make
sudo make install
